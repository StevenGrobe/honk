# honk


Step 1 :  
clone the project to download its contents  
  
```git clone https://github.com/StevenGrobe/honk```

Step 2 :  
make composer install the project's dependecies into vendor/  
  
```cd /honk```  
  
```composer install```  
  
Step 3 :  
host the local server  
  
```symfony server:start```
